# databricks-AI-Functions

The purpose of this repository is to provide a collection of examples demonstrating how to use Databricks AI function for LLM Batch Inference in various scenarios. Each example is designed to be self-contained, making it simple for Data Analysts and Data Engineers to adapt them for their own use cases.
